# 旅途：原神启动！ (Genshin Trip - Launched) 🚀

这是一个基于 HTML5 Canvas 开发的趣味像素风格跑酷小游戏。
不仅致敬了经典的“原神启动”梗，还融入了治愈系的背景与随机生成的挑战关卡。

---

## 🌟 项目特色

* **沉浸式体验**：动态渐变的背景色，随距离变换的剧情文字。
* **双端适配**：完美支持 PC 端键盘操作与移动端触屏虚拟按键。
* **存档系统**：自动记录最高得分，挑战自我极限。
* **神秘彩蛋**：向左走试试？（只是彩蛋~）

---

## 🎮 操作说明

### 💻 PC 端 (键盘)
* **[W] / [空格] / [↑]**：跃起（支持二段跳）
* **[A] / [←]**：向左移动
* **[D] / [→]**：向右移动
* **[P]**：暂停/继续旅途

### 📱 移动端 (触屏)
* **左侧按钮**：控制左右移动
* **右侧“跳”字键**：控制跳跃
* **顶部右侧按钮**：点击可暂停或查看历史记录

---

## 🛠️ 技术实现

* **核心引擎**：纯原生 JavaScript + HTML5 Canvas (无外部库依赖)。
* **物理效果**：实现重力感应、摩擦力模拟及碰撞检测。
* **资源处理**：为了实现单文件运行，音频与图片资源均采用 Base64 编码嵌入。
